# SumVid 隱私權政策

**最後更新日期：2024年**

## 簡介

SumVid（以下簡稱「本擴充功能」）是由我們開發的 Chrome/Edge 瀏覽器擴充功能，用於幫助用戶快速提取 YouTube 影片重點。我們重視您的隱私權，本政策說明我們如何收集、使用和保護您的資訊。

## 資訊收集

### 我們收集的資訊

1. **影片內容資訊**
   - 當您使用本擴充功能時，我們會讀取當前 YouTube 頁面的影片標題、描述和字幕內容
   - 這些資訊僅用於生成影片重點摘要，不會被儲存或分享給第三方

2. **使用統計**
   - 本擴充功能不會收集個人身份識別資訊（如姓名、電子郵件地址等）
   - 我們不會追蹤您的瀏覽歷史或個人偏好

### 我們不收集的資訊

- 個人身份識別資訊
- 瀏覽歷史記錄
- 其他網站的訪問資訊
- 位置資訊
- 設備資訊

## 資訊使用方式

### 我們如何使用您的資訊

1. **生成影片重點**
   - 影片內容會發送到我們的後端服務（部署於 Vercel）
   - 使用 GROQ AI API 分析內容並生成重點摘要
   - 處理完成後，內容不會被保留

2. **時間限制管理**
   - 我們使用 IP 地址來追蹤請求頻率，實施時間限制（每分鐘最多 3 次）
   - IP 地址僅用於限制管理，不會用於其他目的
   - 時間限制數據會在 1 分鐘後自動清除

## 資訊儲存

### 本地儲存

- 本擴充功能使用 Chrome 的本地儲存（chrome.storage.local）來儲存：
  - 時間限制資訊（剩餘生成次數）
  - 這些資訊僅存儲在您的設備上，不會上傳到任何伺服器

### 伺服器儲存

- 我們的後端服務不會永久儲存您的影片內容
- 時間限制快取數據僅在記憶體中臨時保存，不會持久化到資料庫

## 資訊分享

我們**不會**：
- 將您的資訊出售給第三方
- 與廣告商分享您的資訊
- 將您的資訊用於任何商業目的

## 第三方服務

### GROQ AI API

- 我們使用 GROQ AI API 來生成影片重點摘要
- 影片內容會發送到 GROQ 的伺服器進行處理
- 請參考 [GROQ 的隱私權政策](https://groq.com/privacy) 了解他們如何處理數據

### Vercel

- 我們的後端服務部署在 Vercel 平台上
- 請參考 [Vercel 的隱私權政策](https://vercel.com/legal/privacy-policy) 了解他們的數據處理方式

## 資料安全

我們採取以下措施保護您的資訊：

1. **HTTPS 加密**：所有數據傳輸都使用 HTTPS 加密
2. **API 金鑰保護**：API 金鑰存儲在後端服務，不會暴露在前端代碼中
3. **最小化數據收集**：我們只收集必要的資訊來提供服務
4. **自動清理**：時間限制數據會自動過期和清除

## 您的權利

您有權：

1. **停止使用**：隨時可以卸載本擴充功能
2. **清除數據**：卸載擴充功能會清除所有本地儲存的數據
3. **了解數據使用**：本政策說明我們如何使用您的資訊

## 兒童隱私

本擴充功能不針對 13 歲以下的兒童。我們不會故意收集兒童的個人資訊。

## 政策變更

我們可能會不定期更新本隱私權政策。重大變更會通過擴充功能更新通知您。建議您定期查看本政策以了解最新資訊。
